Siena — Tower Drop

Action Sports • Character Dialogue

Introducing Siena—a recurring character built to anchor future reels. The concept: a Lara Croft-type adventurer testing her own board design, launching from an absurdly tall studio tower before cutting to mountain terrain. Primary tools were Kling 2.6 and Kling 01, both of which launched as this project started—they outperformed Veo, Seedance, Wan, and Hailuo on snow physics and realistic body movement. Nano Banana Pro for character development. I invested time building out Siena's look and the studio environment so Kling had enough reference material to keep animations consistent across shots.
